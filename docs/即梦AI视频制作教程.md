### 一、即梦AI提示词万能框架

该框架的核心逻辑是**结构化叠加**，将复杂的创意拆解为7个可控维度，按照固定顺序组合，即可生成高质量提示词。

**万能公式：**
> **风格 + 视角 + 主体 + 背景 + 细节 + 光影 + 质量词**

**各维度详解：**

| 维度 | 作用 | 关键词示例 |
| :--- | :--- | :--- |
| **1. 风格** | 定义画面整体艺术基调与渲染质感 | 高精度建模、UE5质感、水彩风、赛博朋克、写实摄影 |
| **2. 视角** | 决定镜头观看的角度与构图起点 | 正面视角、鸟瞰图、仰视、第一人称、低角度特写 |
| **3. 主体** | 画面的核心描绘对象（谁/什么） | 银灰色敞篷跑车、机械战甲、东方少女、古建筑 |
| **4. 背景** | 主体所处的环境与场景氛围 | 赛博朋克城市夜景、云雾缭绕的山谷、浩瀚星空 |
| **5. 细节** | 画面中的点缀元素，增加丰富度（即"加细节"） | 霓虹灯广告、飘落的花瓣、机械齿轮纹理、雨滴 |
| **6. 光影** | 画面的光线质感与色调氛围（即"加光影"） | 电影感光影、胶片颗粒质感、逆光霞辉、丁达尔效应 |
| **7. 质量词** | 提升画面清晰度与专业度（即"加质量词"） | 8K、大师级构图、高锐度、微距细节、国家地理风格 |

### 二、提示词详解：风格篇

风格是万能公式的第一环，直接奠定画面的基调。深刻理解并运用风格，是提升生成质量和独特性的核心。

#### 1. 风格的重要性
* **影响画风与可控性**：风格直接决定画面的整体视觉走向和AI生成的可控程度。
* **决定视频生成难度**：风格会直接影响后续转化为AI视频的难度。一般而言，难度排序为：**写实 < 3D < 2D动漫**。
* **源于素材数量差异**：上述难度差异的核心，源于AI模型背后训练素材的数量（写实类视频素材最多，因此生成最稳定、难度最低）。
* **打破同质化**：在海量AI内容中，巧妙应用独特或小众的风格，是打破作品同质化、建立个人辨识度的有效手段。

#### 2. 风格的分类
常用的风格提示词主要分为以下三大类：
* **3D类**：3D影视、3A游戏、3D国漫等。
* **绘画类**：油画、素描、游戏美宣图、画师风格（需明确具体画师名或流派）、古风游戏风格等。
* **写实类**：自然摄影、街头摄影、产品摄影、人物摄影等。

#### 3. 风格提示词的获取技巧
* **一般操作**：通过参考优秀的AI绘画作品、摄影网站分类、艺术史流派介绍，积累基础风格词汇。
* **高阶操作**：使用垫图（Image-to-Image）反推提示词，或利用大模型（如 ChatGPT、Claude）描述想要的画面氛围，让AI提炼专业的美术/摄影风格术语，并将这些提炼出的词汇统一存储为 JSON 格式文件（建立个人风格词库），方便后续高效复用。   

#### 4. 风格融合
当单一风格无法满足需求时，可以尝试组合多种风格词（如"赛博朋克 + 浮世绘"）：
* **融合成功的关键**：两种风格之间存在互补关系，在视觉元素或表现逻辑上互不冲突。
* **融合失败的原因**：核心表现手法存在严重冲突（例如要求"极简线条素描"同时又要"极致光影厚涂"），会导致AI解析混乱，生成效果不佳。

### 三、提示词详解：视角篇

视角（镜头控制）是提示词中决定画面构图和观众代入感的关键维度。掌握视角控制，相当于拥有了摄影师的镜头。

#### 1. 视角的概念
* **核心定义**：观察者（镜头）与被摄主体之间的相对位置、方向和距离关系。
* **三个关键维度**：相对位置、方向关系、距离关系。
* **决定因素**：视角会直接影响画面的**透视效果**、**主体形态**以及最终的**叙事氛围**。

#### 2. 镜头的分类
* **按镜头视角分类**：平拍、俯拍、仰拍；鱼眼镜头、长焦镜头、广角镜头。
* **按拍摄高度分类**：高拍（无人机）、低拍（贴地）。
* **按拍摄距离分类**：远景、中景、近景、特写；微距镜头。
* **特殊角度（小众）**：
  * **倾斜镜头**：带来失衡感与视角差。
  * **跟随镜头**：带有运动模糊，包含第一人称的运动感。
  * **过肩镜头**：增强代入感，随从第三者视角。

#### 3. 镜头的实践应用
* **案例一：运动感镜头**
  * **需求**：拍摄人物跑步，体现动感。
  * **方案**：跟随镜头 + 动态模糊 + 背影描述。
  * **效果**：画面更具代入感和张力。
* **案例二：情绪表达**
  * **需求**：表现人物生气。
  * **方案**：特写镜头 + 面部微表情描述。
  * **对比**：如果去掉特写，情绪表达的力度会明显减弱。

#### 4. 经典拍摄视角的应用与词汇
* **三大基础经典视角**：
  1. 过肩视角
  2. 第一人称自拍视角
  3. 第一人称视角（无自拍）
* **进阶视角与构图词汇参考**：
  * **俯拍全景**：上帝视角几何构图 + 云层透视
  * **仰拍特写**：压迫感低角广角畸变
  * **眼睛平视**：第四面墙突破凝视
  * **过肩镜头**：沉浸式 POV 空间包裹感
  * **鸟瞰镜头**：无人机探针视角（螺旋下降）
  * **微距视角**：量子级细节显像（毛孔级纹理）
  * **鱼眼镜头**：曲率空间扭曲入口
  * **窥视构图**：景深隧道窥探框架
  * **倒影镜头**：水面幽冥对称次元
  * **隧道构图**：纵深光轨引导线
  * **框式构图**：建筑切割光影囚笼
  * **分裂屏幕**：平行宇宙叙事蒙太奇

#### 5. 镜头的层次感
为画面增加纵深和高级感，常用以下两招：
* **第一招：物理遮挡型前景**。在镜头前增加真实的物理元素（如植物叶片、门框），配合前景遮挡模糊失焦，能大幅提升画面质感。
* **第二招：氛围粒子型前景**。利用光影、烟雾、雨滴、雪花、飞尘等粒子元素作为前景，烘托场景氛围。

#### 6. 实战进阶：指定构图生成与“三明治”提示词
要实现精准的构图控制，可以通过构建**“三明治”多层信息结构**：
* **维度 1：拍摄者维度的控制锚**
  * 涵盖焦距、光圈、快门、机位高度、相机运动（推/拉/摇/移/手持）。
  * *示例*：85mm人像镜头，光圈 f/1.8，平视角度，三脚架固定。
* **维度 2：被拍摄者维度的控制锚**
  * 涵盖主体位置、朝向、动作幅度、视线方向、与前景/背景的距离。
  * *示例*：年轻女性，正对镜头，肩膀放松，眼神直视，嘴角微扬。
* **维度 3：环境与其他补充**
  * *示例*：浅景深，背景完全虚化成光斑，柔和的侧光打亮脸部，眼神光明显。
* **其他补充方法**：还可以配合使用“排除法”来进一步约束不需要出现的画面元素。

### 四、提示词详解：构图控制篇

构图是画面空间的组织方式，决定了视觉元素的布局和画面的整体表现力。在AI图像和视频生成中，良好的构图是区别普通生成与大师级作品的关键。

#### 1. 构图基础概念
* **传统美术中的重要性**：构图是绘画、摄影等视觉艺术的骨架，决定了作品的视觉张力和审美下限。
* **在AI图像生成中的作用**：构图控制词能引导AI模型合理分配画面空间，避免元素堆砌、主体不明或画面失衡。
* **构图的核心目的**：让画面更完整、更有表现力。
* **定义**：对画面元素进行空间布控、比例分配和关系组织。
* **构图的四大核心目的**：
    1. **突出核心**：让观看者第一眼聚焦画面主体，避免杂乱无章，确保视觉中心明确。
    2. **建立秩序**：通过位置、大小、色彩搭配让画面协调统一，建立视觉上的逻辑感。
    3. **引导视线**：利用线条、光影或元素排列引导观众的阅读顺序，形成视觉流动。
    4. **强化表达**：通过特定的构图强化情绪氛围，暗示角色身份地位或故事走向（例如低视角仰拍暗示强大，高视角俯拍暗示渺小）。

#### 2. 构图的核心要素
构图不仅仅是摆放位置，而是多维度的空间关系处理：
* **元素排布（Composition）**：决定主体、陪体、前景、背景在画面二维平面上的具体位置（如左、右、居中、边缘）。
* **比例分割（Proportion/Division）**：画面空间的划分方式，如三分法（黄金分割）、对半分割、大面积留白与小面积主体的对比等。
* **视觉平衡（Visual Balance）**：通过元素的重量感（大小、色彩深浅、密度）在画面中实现对称平衡或非对称的动态平衡。
* **空间关系（Spatial Relationship）**：利用透视、遮挡、虚实变化，在二维画面中营造出三维的深度感和层次感。

#### 3. 常见构图类型详解
掌握经典构图模型，能快速建立画面的高级感：
* **中心构图 (Center Composition)**：主体居中，突出明确，但略显普通和刻板。（示例：梵高《向日葵》）
* **对称构图 (Symmetrical Composition)**：左右或上下对称，呈现稳定、正式、庄重、神圣的美感。（示例：拉斐尔《雅典学院》）
* **对角线构图 (Diagonal Composition)**：利用线条将画面沿对角线分割，增强动感、活力或视觉冲突。（示例：安格尔《大宫女》）
* **引导线构图 (Leading Lines)**：借助画面中的直线或曲线（如道路、河流、建筑边缘）引导视线指向主体。（示例：梵高《星月夜》）
* **框架构图 (Framing Composition)**：利用门、窗、树枝等前景元素形成框架，将主体框在其中，增加深度和窥视感。（示例：塞尚的风景画）
* **三角形构图 (Triangle Composition)**：主体或多个元素构成三角形（正三角或倒三角），形成极强的稳定感和坚不可摧的视觉印象，轮廓近似即可。（示例：安格尔《德布罗意公主》）
* **散点构图 (Scattered Composition)**：相似元素在画面中重复出现，错落有致，“形散神不散”，常用于表现群像或特定氛围。（示例：修拉《大碗岛的星期天下午》）
* **留白构图 (Negative Space/Minimalist)**：画面中保留大量空白空间，营造空灵意境、极简主义或强化主体的孤独感。（示例：八大山人水墨画）
* **黄金螺旋构图 (Golden Spiral/Fibonacci)**：遵循斐波那契数列的螺旋线构图，视觉中心位于螺旋焦点，是最符合人类审美的自然比例。（示例：达芬奇《蒙娜丽莎》）

#### 4. 构图的灵活运用
* **并非死板规则**：构图知识是前人经验的总结，并非必须严格遵从的铁律。
* **好看即合理**：只要画面本身视觉效果出色，能够准确传达意图，就是合理的构图。
* **作为辅助工具**：将构图知识作为“检测”和“修改”画面的辅助工具，当画面感觉不对劲时，用构图理论去诊断问题。
* **与视角密不可分**：构图对于视频分镜至关重要，且通常与“视角”（镜头角度）结合使用，两者共同决定画面的空间结构。

#### 5. 实战技巧：如何在提示词中控制构图
* **直接描述构图名词**：最简单有效的方式，如直接输入 `三分法构图`、`中心对称构图`、`对角线构图`。
* **客观描述元素位置关系**：当 AI 无法准确理解抽象的构图名词时，改用客观的方位描述。例如：不写 `框架构图`，改写为 `前景是深色的窗框，透过窗户看到远处的主体`。
* **综合描述**：结合拍摄角度、留白、剪影等元素进行综合描述。例如：`低角度仰拍，右下角是大面积的深色岩石剪影，左上方留白展示璀璨星空`。
* **即梦基础改图**：利用即梦AI的局部重绘或扩图功能，在生成后进行二次构图调整。

#### 6. 延伸学习与高阶控制
* **进阶内容**：探索更复杂的构图方式，如对点构图、黄金螺旋构图的变体等。
* **资料的获取方式**：多从优秀的摄影作品、电影截图、经典绘画中汲取灵感，并尝试用语言解构其构图方式。
* **示意图控制构图（ControlNet等辅助功能）**：
    * **文字标示法**：在参考图中用文字直接标注元素位置。
    * **色块标示法**：用不同颜色的简单色块画出构图草图，作为控制底图让 AI 生成（如使用 Depth、Scribble 等控制模型）。

### 五、套用框架的提示词案例（3个新案例）

以下案例严格按照 **"风格+视角+主体+背景+细节+光影+质量词"** 的顺序编写：

#### 案例 1：国风水墨武侠
> **提示词：** 新中式水墨风格，宣纸纹理质感，侧面仰视视角，一位身穿蓑衣头戴斗笠的孤独剑客，站立于悬崖孤松之上，远处是云雾缭绕的层叠群山，衣袂随风飘动，松枝上落有积雪，月光透过云层洒下清冷银辉，画面留白意境深远，4K超清，大师级水墨技法，电影级景深。

#### 案例 2：末日机甲废墟
> **提示词：** 写实科幻风格，3D渲染细节，低角度仰视视角，一台伤痕累累的橙黄色巨型机甲战士，单膝跪在废弃都市的废墟中央，背景是核爆后暗红色的蘑菇云与倒塌的高楼大厦，机甲表面有弹痕和裸露的电线，周围弥漫着硝烟与火星，暖色调的爆炸逆光勾勒出机甲轮廓，8K超写实，极致光影，工业光魔级特效。

#### 案例 3：奇幻精灵微距
> **提示词：** 梦幻童话风格，微距摄影质感，平视近距离视角，一只通体透蓝的玻璃翅膀小精灵，坐在巨大的绿色蕨类植物叶片上，背景是繁星点点的深邃森林夜空，叶片上挂着晶莹的露珠，精灵周围有细小的金色光点环绕飞舞，柔和的侧逆光穿透精灵的翅膀呈现半透明效果，超细节，8K，壁纸级构图，色彩艳丽。

### 六、使用小贴士
- **顺序不可乱**：即梦AI对按此顺序输入的提示词响应最精准。
- **"加"字诀**：在"细节"、"光影"、"质量词"这三个模块中，可以大胆使用形容词叠加（如你原图中的"加细节加光影再加质量词"），AI会自动理解并丰富对应维度。
