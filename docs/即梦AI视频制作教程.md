### 一、即梦AI提示词万能框架

该框架的核心逻辑是**结构化叠加**，将复杂的创意拆解为7个可控维度，按照固定顺序组合，即可生成高质量提示词。

**万能公式：**
> **风格 + 视角 + 主体 + 背景 + 细节 + 光影 + 质量词**

**各维度详解：**

| 维度 | 作用 | 关键词示例 |
| :--- | :--- | :--- |
| **1. 风格** | 定义画面整体艺术基调与渲染质感 | 高精度建模、UE5质感、水彩风、赛博朋克、写实摄影 |
| **2. 视角** | 决定镜头观看的角度与构图起点 | 正面视角、鸟瞰图、仰视、第一人称、低角度特写 |
| **3. 主体** | 画面的核心描绘对象（谁/什么） | 银灰色敞篷跑车、机械战甲、东方少女、古建筑 |
| **4. 背景** | 主体所处的环境与场景氛围 | 赛博朋克城市夜景、云雾缭绕的山谷、浩瀚星空 |
| **5. 细节** | 画面中的点缀元素，增加丰富度（即"加细节"） | 霓虹灯广告、飘落的花瓣、机械齿轮纹理、雨滴 |
| **6. 光影** | 画面的光线质感与色调氛围（即"加光影"） | 电影感光影、胶片颗粒质感、逆光霞辉、丁达尔效应 |
| **7. 质量词** | 提升画面清晰度与专业度（即"加质量词"） | 8K、大师级构图、高锐度、微距细节、国家地理风格 |

### 二、提示词详解：风格篇

风格是万能公式的第一环，直接奠定画面的基调。深刻理解并运用风格，是提升生成质量和独特性的核心。

#### 1. 风格的重要性
* **影响画风与可控性**：风格直接决定画面的整体视觉走向和AI生成的可控程度。
* **决定视频生成难度**：风格会直接影响后续转化为AI视频的难度。一般而言，难度排序为：**写实 < 3D < 2D动漫**。
* **源于素材数量差异**：上述难度差异的核心，源于AI模型背后训练素材的数量（写实类视频素材最多，因此生成最稳定、难度最低）。
* **打破同质化**：在海量AI内容中，巧妙应用独特或小众的风格，是打破作品同质化、建立个人辨识度的有效手段。

#### 2. 风格的分类
常用的风格提示词主要分为以下三大类：
* **3D类**：3D影视、3A游戏、3D国漫等。
* **绘画类**：油画、素描、游戏美宣图、画师风格（需明确具体画师名或流派）、古风游戏风格等。
* **写实类**：自然摄影、街头摄影、产品摄影、人物摄影等。

#### 3. 风格提示词的获取技巧
* **一般操作**：通过参考优秀的AI绘画作品、摄影网站分类、艺术史流派介绍，积累基础风格词汇。
* **高阶操作**：使用垫图（Image-to-Image）反推提示词，或利用大模型（如 ChatGPT、Claude）描述想要的画面氛围，让AI提炼专业的美术/摄影风格术语，并将这些提炼出的词汇统一存储为 JSON 格式文件（建立个人风格词库），方便后续高效复用。   

#### 4. 风格融合
当单一风格无法满足需求时，可以尝试组合多种风格词（如"赛博朋克 + 浮世绘"）：
* **融合成功的关键**：两种风格之间存在互补关系，在视觉元素或表现逻辑上互不冲突。
* **融合失败的原因**：核心表现手法存在严重冲突（例如要求"极简线条素描"同时又要"极致光影厚涂"），会导致AI解析混乱，生成效果不佳。

### 三、提示词详解：视角篇

视角（镜头控制）是提示词中决定画面构图和观众代入感的关键维度。掌握视角控制，相当于拥有了摄影师的镜头。

#### 1. 视角的概念
* **核心定义**：观察者（镜头）与被摄主体之间的相对位置、方向和距离关系。
* **三个关键维度**：相对位置、方向关系、距离关系。
* **决定因素**：视角会直接影响画面的**透视效果**、**主体形态**以及最终的**叙事氛围**。

#### 2. 镜头的分类
* **按镜头视角分类**：平拍、俯拍、仰拍；鱼眼镜头、长焦镜头、广角镜头。
* **按拍摄高度分类**：高拍（无人机）、低拍（贴地）。
* **按拍摄距离分类**：远景、中景、近景、特写；微距镜头。
* **特殊角度（小众）**：
  * **倾斜镜头**：带来失衡感与视角差。
  * **跟随镜头**：带有运动模糊，包含第一人称的运动感。
  * **过肩镜头**：增强代入感，随从第三者视角。

#### 3. 镜头的实践应用
* **案例一：运动感镜头**
  * **需求**：拍摄人物跑步，体现动感。
  * **方案**：跟随镜头 + 动态模糊 + 背影描述。
  * **效果**：画面更具代入感和张力。
* **案例二：情绪表达**
  * **需求**：表现人物生气。
  * **方案**：特写镜头 + 面部微表情描述。
  * **对比**：如果去掉特写，情绪表达的力度会明显减弱。

#### 4. 经典拍摄视角的应用与词汇
* **三大基础经典视角**：
  1. 过肩视角
  2. 第一人称自拍视角
  3. 第一人称视角（无自拍）
* **进阶视角与构图词汇参考**：
  * **俯拍全景**：上帝视角几何构图 + 云层透视
  * **仰拍特写**：压迫感低角广角畸变
  * **眼睛平视**：第四面墙突破凝视
  * **过肩镜头**：沉浸式 POV 空间包裹感
  * **鸟瞰镜头**：无人机探针视角（螺旋下降）
  * **微距视角**：量子级细节显像（毛孔级纹理）
  * **鱼眼镜头**：曲率空间扭曲入口
  * **窥视构图**：景深隧道窥探框架
  * **倒影镜头**：水面幽冥对称次元
  * **隧道构图**：纵深光轨引导线
  * **框式构图**：建筑切割光影囚笼
  * **分裂屏幕**：平行宇宙叙事蒙太奇

#### 5. 镜头的层次感
为画面增加纵深和高级感，常用以下两招：
* **第一招：物理遮挡型前景**。在镜头前增加真实的物理元素（如植物叶片、门框），配合前景遮挡模糊失焦，能大幅提升画面质感。
* **第二招：氛围粒子型前景**。利用光影、烟雾、雨滴、雪花、飞尘等粒子元素作为前景，烘托场景氛围。

#### 6. 实战进阶：指定构图生成与“三明治”提示词
要实现精准的构图控制，可以通过构建**“三明治”多层信息结构**：
* **维度 1：拍摄者维度的控制锚**
  * 涵盖焦距、光圈、快门、机位高度、相机运动（推/拉/摇/移/手持）。
  * *示例*：85mm人像镜头，光圈 f/1.8，平视角度，三脚架固定。
* **维度 2：被拍摄者维度的控制锚**
  * 涵盖主体位置、朝向、动作幅度、视线方向、与前景/背景的距离。
  * *示例*：年轻女性，正对镜头，肩膀放松，眼神直视，嘴角微扬。
* **维度 3：环境与其他补充**
  * *示例*：浅景深，背景完全虚化成光斑，柔和的侧光打亮脸部，眼神光明显。
* **其他补充方法**：还可以配合使用“排除法”来进一步约束不需要出现的画面元素。

### 四、套用框架的提示词案例（3个新案例）

以下案例严格按照 **"风格+视角+主体+背景+细节+光影+质量词"** 的顺序编写：

#### 案例 1：国风水墨武侠
> **提示词：** 新中式水墨风格，宣纸纹理质感，侧面仰视视角，一位身穿蓑衣头戴斗笠的孤独剑客，站立于悬崖孤松之上，远处是云雾缭绕的层叠群山，衣袂随风飘动，松枝上落有积雪，月光透过云层洒下清冷银辉，画面留白意境深远，4K超清，大师级水墨技法，电影级景深。

#### 案例 2：末日机甲废墟
> **提示词：** 写实科幻风格，3D渲染细节，低角度仰视视角，一台伤痕累累的橙黄色巨型机甲战士，单膝跪在废弃都市的废墟中央，背景是核爆后暗红色的蘑菇云与倒塌的高楼大厦，机甲表面有弹痕和裸露的电线，周围弥漫着硝烟与火星，暖色调的爆炸逆光勾勒出机甲轮廓，8K超写实，极致光影，工业光魔级特效。

#### 案例 3：奇幻精灵微距
> **提示词：** 梦幻童话风格，微距摄影质感，平视近距离视角，一只通体透蓝的玻璃翅膀小精灵，坐在巨大的绿色蕨类植物叶片上，背景是繁星点点的深邃森林夜空，叶片上挂着晶莹的露珠，精灵周围有细小的金色光点环绕飞舞，柔和的侧逆光穿透精灵的翅膀呈现半透明效果，超细节，8K，壁纸级构图，色彩艳丽。

### 五、使用小贴士
- **顺序不可乱**：即梦AI对按此顺序输入的提示词响应最精准。
- **"加"字诀**：在"细节"、"光影"、"质量词"这三个模块中，可以大胆使用形容词叠加（如你原图中的"加细节加光影再加质量词"），AI会自动理解并丰富对应维度。
